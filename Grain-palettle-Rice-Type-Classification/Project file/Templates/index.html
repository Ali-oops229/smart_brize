About the Rice Type Classification Model

This model is built using Convolutional Neural Network and trained on labelled images of rice.

  • Accuracy: Predicts correctly 97% of the time.
  • Varieties Detected: 5 different rice types.
  • Dataset Source: Kaggle rice image dataset.
  • Model Architecture: MobileNetV4 with transfer learning.
